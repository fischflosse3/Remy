# Remy

Lade diesen Ordner in Chrome über `chrome://extensions` als entpackte Erweiterung.

Für echte KI-Antworten muss das Backend laufen. Die technische Verbindung ist in der UI ausgeblendet.
Wenn das Backend nicht läuft, nutzt das Popup automatisch die lokale Fallback-Suche.

## Version 0.5.0

Verbesserte Unterstützung für dynamische Seiten:
- YouTube-Videos und YouTube-Suchen
- Twitch-Metadaten
- Netflix-Metadaten, soweit sichtbar
